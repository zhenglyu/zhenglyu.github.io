官方解释有些繁琐，简单的解释是：Windows To Go （后面简称 WTG）可以将 Windows系统安装到一台 USB驱动器中（例如 U盘、移动硬盘等）。
这个安装了 Windows 的驱动器，可以在任何一台没有安装 Windows 的设备中运行起 Windows 。
